개발하면서 알게 되는 지식들이나 정보들을 요약해 저장해 놓으면 나중에 필요할 때 활용하기 편할 것 같다 생각해 블로그를 찾아보았다 만
네이버는 글 작성하기가 귀찮아지는 UI라 포기.. velog나 tistory 중 고민하다 우연하게 Obsidian의 CEO인 kepano의 [블로그 포스트를 보게 되었다](https://www.threads.com/@kepano/post/DDxmyWzOKp0?xmt=AQGzbVXXjv9Q0CIid-bRmSBo1EDEpnRtGOVVW1_YdLxkHw).

해당 블로그의 핵심은 Obsidian과 Jekyll을 활용한 포스팅 및 git push를 사용한 배포. 별도의 사이트 접속 필요없이 git clone한 repo를 작성하고 push하면 내용이 업데이트되기에 pull/push를 제외하고는 local로 작성/저장이 가능한 장점이 있다. 학교
